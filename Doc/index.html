Gerenciamento de Turmas

O sistema de gerenciamento de turmas permite a criação, edição e exclusão de turmas, além da visualização de alunos matriculados em cada turma.

Funcionalidades principais:

  • Cadastro de Turmas: Você pode cadastrar novas turmas, informando dados como nome, código e horário.
  • Visualização das Turmas: Exiba a lista de turmas já cadastradas e acesse detalhes sobre cada uma, incluindo os alunos matriculados.
  • Matricular Alunos: Matricule alunos em turmas específicas para gerenciar suas inscrições e participações.
  • Excluir Turmas: Caso uma turma não seja mais necessária, é possível removê-la do sistema, excluindo todos os dados relacionados.

Para gerenciar as turmas, acesse as opções acima para realizar as operações desejadas. A plataforma permite que os administradores mantenham as turmas organizadas e assegurem que os alunos estejam devidamente matriculados.
